# 🚀 EPIC UNIVERSE MISSION - DEPLOYMENT GUIDE

## PART 1: GITHUB SETUP (5 minutes)

### Step 1: Create the Repository on GitHub
1. Go to https://github.com/new
2. **Repository name:** `epic-universe-mission`
3. **Description:** "Operation Epic Universe - Interactive Mission Briefing"
4. **Visibility:** Public
5. **Initialize:** Leave unchecked (we already have files)
6. Click **Create Repository**

### Step 2: Push Code to GitHub
1. Extract the project folder to your PC
2. Open a terminal/command prompt in the project folder
   - **Windows:** Right-click folder → "Open in Terminal" or "Open PowerShell here"

3. **If on Windows:** Run:
   ```
   setup-github.bat
   ```

4. When prompted for password, paste your GitHub token (REDACTED):
   ```
   [REDACTED]
   ```

5. Wait for it to say "Branch 'main' set up to track remote branch 'main'"

### Step 3: Enable GitHub Pages
1. Go to https://github.com/superswiss/epic-universe-mission
2. Click **Settings** (top right)
3. Left sidebar → **Pages**
4. Under "Build and deployment":
   - **Source:** GitHub Actions

### Step 4: Your App is Live! 🎉
Visit: **https://superswiss.github.io/epic-universe-mission**
