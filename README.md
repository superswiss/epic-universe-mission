# Operation Epic Universe - Mission Briefing

Interactive mission briefing app for coordinating your Epic Universe trip!

## Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Google Sheets (See SETUP_GUIDE.md)
Create a `.env.local` file with your credentials:
```
VITE_GOOGLE_SHEET_ID=your-sheet-id
```

### 3. Run Locally
```bash
npm run dev
```

### 4. Deploy
```bash
npm run deploy
```
